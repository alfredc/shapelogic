Imageutil

Classes for images, image operations and abstraction of images.

This contains a abstraction over 2 main image types are are currently supported.
  • ImageProcessor from ImageJ.
  • BufferedImage from Java.
It also contain other abstract image concepts that are used by image processing, but not doing image processing.

Dependency

  • This package can depend on util, calculation, streams.
  • There is a dependency of any ImageJ ImageProcessor, but no other external libraries outside Java 6.
  • This can be used in a Java application, just by adding shapelogic_.jar, ij.jar is only needed if you use the ImageJ images.
