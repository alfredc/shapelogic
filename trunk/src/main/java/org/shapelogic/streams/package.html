Streams

New library for functional programming style streams / lazy lists.

Dependency

  • This package can depend on util, calculation, scripting.
  • There is no dependency of any image processing code or any external libraries outside Java 6.
  • This can be used in a Java application, just by adding shapelogic_.jar.

Declarative programming

Declarative programming in ShapeLogic is build on the following:

  • Rule database that can access Java object using Scripting languages or JEXL
  • Streams / lazy lists
  • Lazy calculations
  • Goal driven tasks
  • Maybe later IoC

Streams

Definitions

last the last element of a stream, set to LAST_UNKNOWN, before it is know. This value can get lower as calculation progress.

maxLast maually set max value for last. The real last value might be lower.

How can an calculation of stream end?

  • If null is not a legal value when finding a null
  • If an exception is thrown during a calculation
  • By setting a top maxLast value
